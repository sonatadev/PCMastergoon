# PCMastergoon 🖥️

A full-stack web application built with PHP, HTML, CSS, and JavaScript.
The project simulates a basic e-commerce system with user authentication, product browsing, shopping cart, and order management.

---

## 🚀 Features

* User registration and login system
* Secure password hashing with `password_hash()` and `password_verify()`
* Product catalog and product detail pages
* Shopping cart functionality
* Checkout system
* Order history and order details
* Admin/management sections (if applicable)
* Responsive frontend using HTML, CSS, and JavaScript

---

## 🔐 Security

* Passwords are securely hashed using PHP built-in hashing functions
* SQL queries use prepared statements to prevent SQL injection
* Session-based authentication system

---

## 🛠️ Tech Stack

* PHP (backend logic)
* MySQL (database)
* HTML5 / CSS3 (frontend structure & styling)
* JavaScript (frontend interactions)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/sonatadev/PCMastergoon.git
```

2. Import the database into MySQL

3. Configure database connection in `db.php`

4. Run the project using a local server (XAMPP / LAMP / Nginx + PHP)

5. Open in browser:

```
http://localhost/PCMastergoon
```
